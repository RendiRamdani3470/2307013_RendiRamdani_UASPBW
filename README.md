###  Fitur yang Berhasil Diimplementasikan:
1. **Firebase Auth**: Sistem Login & Register yang aman untuk setiap user.
2. **Google Books API**: Pencarian jutaan buku secara real-time dan akurat.
3. **Advanced UI/UX**: Tampilan Dark Mode modern dengan layout sidebar yang responsif dan interaktif.
4. **Multi-Collection CRUD**: 
   - **Favorit**: Menyimpan buku yang disukai ke database Cloud Firestore.
   - **Baca Nanti**: Fitur bookmark untuk menandai buku yang ingin dibaca di masa depan.
5. **Instant Read**: Tombol akses langsung untuk membaca/preview isi buku di tab baru.
6. **Real-time Synchronization**: Data favorit dan daftar bacaan terupdate otomatis tanpa perlu refresh halaman.

---

### Tech Stack:
* **Frontend**: React.js (Vite)
* **Backend & Database**: Cloud Firestore (Firebase)
* **Authentication**: Firebase Auth
* **HTTP Client**: Axios

---

###  Cara Menjalankan Project:
1. Clone repository ini.
2. Jalankan perintah `npm install` untuk menginstall library yang dibutuhkan.
3. Jalankan perintah `npm run dev` untuk memulai server lokal.
4. Buka browser di alamat yang tertera (biasanya `localhost:5173`).
